# @react-error-utils/testing

## 1.0.0

### Minor Changes

- Initial release of react-error-utils monorepo

  Comprehensive React error handling library with:

  - ErrorBoundary component with fallback support
  - useErrorHandler hook for propagating errors
  - useRetry hook for retrying failed operations
  - React 19 use() hook integration
  - Testing utilities for error boundary testing
  - Error reporting integrations (Sentry, Bugsnag)

### Patch Changes

- Updated dependencies
  - @react-error-utils/core@0.1.0
